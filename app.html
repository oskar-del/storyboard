node mcp-server.js
2. Add to Claude Desktop config
~/Library/Application Support/Claude/
claude_desktop_config.json
"storyboard": { "command": "node",
  "args": ["/full/path/to/mcp-server.js"] }
Restart Claude Desktop. Sessions start appearing here automatically.

Install browser extension
Capture Claude.ai, ChatGPT, Gemini, Perplexity automatically — no terminal needed.
→
3 steps, 2 minutes
1. Open Chrome → chrome://extensions
2. Enable Developer mode (top right toggle)
3. Click Load unpacked → select the browser-extension/ folder
Also start the MCP server: node mcp-server.js
Then visit Claude.ai — your session will appear here within 30 seconds.
